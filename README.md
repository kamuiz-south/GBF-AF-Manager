## 🛡️ 安全性と仕組みについて

### AF Collector (ブラウザ拡張機能)
- **自己責任での利用**: ブラウザ拡張機能を使用してゲーム画面からデータを収集する行為は、ゲームの利用規約に照らしてご自身の判断で行ってください。
- **非干渉の設計**: 本機能は「表示されている情報を読み取る」ことのみに特化しています。**ゲームデータの改ざん、自動操作（マクロ）、通信の偽装などは一切行いません。**
- **開発者モード対応**: より安全に作業したい方向けに、F12の開発者モード（ネットワークログ）からの手動インポート機能もネイティブでサポートしています。

### AF Manager (デスクトップ版 / Web版)
- **完全な独立性**: Managerアプリ自体はゲームブラウザとは完全に別のプロセスで動作します。ゲームの挙動に一切干渉しないため、安全にご利用いただけます。

## 🛠 導入方法

### 拡張機能（AF Collector）
1. `AF_Collector_1.0.0.zip` を解凍します。
2. ブラウザの拡張機能管理画面 (`chrome://extensions/`) を開きます。
3. 「デベロッパーモード」をONにします。
4. 「展開して読み込む」から解凍したフォルダを選択します。

### デスクトップアプリ（AF Manager）
- インストーラー版、またはポータブル版のいずれかお好みのものを実行してください。
